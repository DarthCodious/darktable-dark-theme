# darktable-dark-theme
A custom dark mode CSS theme for darktable, an open source image editor available on Windows, MacOS, and Linux! This custom CSS theme works for me on all 3 platforms. 

Instructions: once you've downloaded and installed darktable, simply navigate to the darkroom tab, then click the settings gear icon on the top right. 

Go to the general tab. 

Change the theme to "darktable". 

Then, Copy your existing code and paste it into a notepad doc. Keep it safe as a backup. Then, paste the new CSS code from the file in this github repo (it is called "darktable dark theme CSS").  

Enjoy the dark side!
